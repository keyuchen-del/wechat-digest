# 公众号推送整理与总结助手

## 本次重点优化

1. **UI 重构**：改为接近飞书多维表格风格的简洁信息面板布局（浅色背景、卡片化、清晰层级）。
2. **抓取提速与稳定性**：新增超时控制、双通道抓取（代理失败自动回退直连）、公众号批量并发抓取与去重入库。

## 使用

- 输入文章地址，点击「抓取该文章」自动整理入库。
- 输入公众号名称，点击「抓取最近文章」并发整理最近候选文章（实验能力，受目标站点策略影响）。
- 在文章库筛选后，选择目标文章，输入自然语言生成总结。

## 部署

推荐 Vercel：导入仓库后直接 Deploy（仓库已包含 `vercel.json`）。
