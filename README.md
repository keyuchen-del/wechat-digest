# 公众号推送整理与总结助手

## 本次重点优化

1. **字体更美观**：升级为更接近飞书风格的字体栈与字体渲染设置（中文/英文混排更清晰）。
2. **解决 `Failed to fetch` 高频问题**：修复代理 URL 拼接错误，并增加更明确的网络拦截提示。
3. **新增功能：公众号文章列表**：输入公众号名称后，先展示候选文章列表与原文链接，并提供“入库”按钮按需抓取。

## 使用

- 输入文章地址，点击「抓取该文章」自动整理入库。
- 输入公众号名称，点击「获取文章列表」，会返回文章标题和链接。
- 在文章列表中可直接点开原文，或点击「入库」导入到文章库后做总结。

## 部署

推荐 Vercel：导入仓库后直接 Deploy（仓库已包含 `vercel.json`）。
